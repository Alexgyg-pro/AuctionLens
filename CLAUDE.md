# AuctionLens — Instructions pour Claude

## Brique eVision (reconnaissance d'image)

Le dossier `evision/` contient un composant React autonome et terminé.
Il ne doit pas être modifié, refactoré, ni réécrit.

Il sera intégré tel quel dans l'application AuctionLens en tant que
composant `<ImageRecognizer>` côté acheteur.

Voir `evision/README.md` pour la documentation complète et les props.

## Stack cible

- Frontend : React + Vite
- Backend : Node.js / Express
- Base de données : SQLite (locale, provisoire)
- Pas d'app mobile native — site responsive uniquement
